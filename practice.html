Beginner - Practice makes perfect!

- Intan Shahirah

You can start draw with the very basic round shapes. Repeat the shapes with different sizes. After that you can go with spiral shapes. Keep on repeat the drawing. You also can use other lines and object and keep on doing the repetition with it.

By doing shapes, you can create some object that pops into your head. With only one shapes, try to create as much doodle you can do! You can also search for the inspiration doodle shape by others.

Try to create the cute doodle mix from a few shapes. For example, cloud and vertical rectangle can create a cute cupcake. Try to create the best doodle version of you!

Not just drawing some objects. You can also use variety of unique typography for doodle. This can make you practice the using of lines in drawing.
